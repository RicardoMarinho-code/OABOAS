# Plataforma de Estudos OAB

Um aplicativo web gamificado para estudos da prova da OAB, inspirado na interface do Duolingo.

Desenvolvido com [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), [Next.js](https://nextjs.org/), [Tailwind](https://tailwindcss.com/), e [Zustand](https://github.com/pmndrs/zustand). O projeto foi inicializado com [create-t3-app](https://github.com/t3-oss/create-t3-app).
